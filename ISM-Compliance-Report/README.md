# ISM-Compliance-Report
This Document is a combination of two reports, an ISO 27001 information system report on AXA financials that defines the scope of the organisation as regards to security, develops an organisational structure to incorporate information security management process, to provide standards as a guideline to strengthen the report and security procedures of the organisation and defines the company for security management. The second report is a compliance report for a fictional institution called dublin general hospital to identify the crown jewel of the institution, their organisational structure, data handling components, GDPR regulations as regards their various data handling processes and neccesseties and to provide non-compliance checking methods and recommendations on how to harden their overall security architecture.
